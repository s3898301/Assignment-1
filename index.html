Job

This project will be a network simulator tutorial which is aimed to help residents set up their network at home optimally to ensure coverage and understanding of devices like routers.

This idea is interesting as from experience, I have had to gather information all by myself on the web to ensure that my wifi was working to the best. I want to ensure that people will be educated and know what will truly help their home networks.

The product will be an interactive object oriented environment where users can select and connect devices, there will be descriptions and error checks which will help users build their own network and see the outcome. Indicators such as speed may be included to represent interaction and connection of devices.

This software will only require a computer and program like visual studio to develop the app. This team will require programming and art. If this project is successful, users will be able choose what products to buy and what to use but also set up their home network by themselves.
